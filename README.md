# 酱香微生物分析-量子方案（QVAE/BM · 开物 Kaiwu 1.3.1）

对接"科技攻关·揭榜挂帅"第六期 **04 号榜题**：*酱香型白酒高温大曲抑制乳酸菌的接触与非接触效应解析研究*。
用玻色量子已在单细胞工作 **scQuantaVita** 中验证的"量子‑经典混合玻尔兹曼能量模型"范式，
把菌群多组学的"关键微生物群、谁抑制谁、接触还是非接触、贡献多少"变成一套可计算、可读出、可湿实验验证的模型量。
**求解器全部使用开物 Kaiwu SDK（公开版 1.3.1），无任何外置自写求解器。**

## 核心思路：两步走（QVAE 筛选 → BM 互作分析）

```
多组学(几百~上千特征)  ──①QVAE 筛选──►  少数关键菌群/代谢物  ──②BM 互作分析──►  带符号互作网络 J
                                                                              ├─ 接触 / 非接触
                                                                              ├─ 广谱 / 特异
                                                                              └─ 各通路对乳酸菌丰度的贡献度
```

1. **① QVAE 负责"筛选"**：用 QVAE（DVAE+RBM）把高维、稀疏的菌群多组学压成低维表示，并给每个样本一个"群落系统势 CSP（能量）"；据此从成百上千个特征里**抓出少数关键菌群 / 代谢物**——判据是"丰度随发酵阶段能量面(CSP)迁移最相关（演替中心菌）"**并入**"与关键乳酸菌 LAB 直接关联的菌"，同时排除噪声特征。
2. **② BM 负责"互作分析"**：**只在筛出的关键菌群**上训练玻尔兹曼机（BM），学出它们之间的**带符号直接互作网络 J**——由此判定 **接触 / 非接触**（直接耦合 vs 经代谢物中介）、**广谱 / 特异**，并用反事实敲除量化各通路对乳酸菌丰度的**贡献度**。

**思路分两步**：先用 QVAE 把问题降到"关键少数"，BM 再在这个可控子网上做严格的直接耦合分析。

## Tutorial 会证明什么（合成数据预埋真值，可复现；全程 Kaiwu 1.3.1）
1. **QVAE + 能量面筛选**：项目内 DVAE+RBM 学表示与群落系统势 CSP（随发酵阶段迁移）；按"丰度随能量面迁移 + 与 LAB 关联"从含 20 噪声的 27 维表干净选出 7 个结构特征（LAB/C/NC/M/BS/co1/T2），排除全部噪声。
2. **玻尔兹曼机学互作网络** {J,h}：负相由 **Kaiwu SA 采样 + e^(−E) 重加权**估计（伊辛矩阵由 `kw.conversion` 生成）。
3. **接触 vs 非接触**（榜题核心）：
   - `C`：直接负耦合 `J[C,LAB]≈-0.39` ⇒ **接触型·特异**
   - `NC`：直接 `J[NC,LAB]≈-0.09`≈0，但 `J[NC,M]>0, J[M,LAB]<0` ⇒ **非接触/经代谢物 M 介导**
   - ✓ 玻尔兹曼机扣除间接相关——**相关≠互作**
4. **广谱 vs 特异**：`BS`=广谱（LAB/co1/T2 多靶）、`C`/`NC`=特异。
5. **贡献度 + ≥60% 关键菌**：反事实敲除（Kaiwu 重采样）量化各通路对 LAB 丰度贡献，累计 ≥60% 关键抑制菌集 `{BS, M}`（≈65%）。
