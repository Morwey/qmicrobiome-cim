## 0. 一页纸结论

榜题的科学核心是一句话：**在酱酒固态发酵的多组学时空数据里，判定高温大曲带入酒醅的哪些微生物/代谢物在"直接"或"通过代谢物间接"地压制关键乳酸菌，并给每条通路一个可量化、可验证的贡献度。**

这本质上是一个**从大量样本反推"谁和谁直接互作"的逆问题**——在统计物理里就是**逆 Ising / 玻尔兹曼机学习（inverse Ising / Boltzmann Machine, BM）**。我们提出的路线，正是把玻色量子已在 *Science* 级单细胞工作 **scQuantaVita** 中验证过的"**量子‑经典混合 = 相干伊辛机（CIM）做玻尔兹曼采样 + 经典网络训练**"范式，迁移到菌群多组学：

```
多组学丰度表 ──①QVAE(玻尔兹曼态编码)──► 离散隐层 + 关键菌群/代谢物
                                              │
                                              ▼
                          ②玻尔兹曼机互作层（CIM 采样训练）
                                       学出耦合网络 {J_ij, h_i}
                                              │
              ┌───────────────┬──────────────┴───────────────┐
              ▼               ▼                              ▼
   接触/非接触判据      广谱/特异 & 长/短           ③反事实敲除(CIM 重采样)
   (直接耦合 vs 中介路径)   (耦合度/时序)              → 每条通路对乳酸菌丰度的贡献度 ≥60% 关键菌
```

**为什么必须上量子**：训练一个**全连接、含"阻挫"（frustration，即竞争性负耦合）**的玻尔兹曼机，瓶颈在负相采样——需要从模型分布 `p(s)∝e^{-E(s)}/Z` 采样，配分函数 `Z=Σ_s e^{-E(s)}` 随节点数指数爆炸，经典 MCMC 在稠密、阻挫的生态网络上收敛慢且有偏。CIM 用简并光参量振荡器（DOPO）作自旋，天然对伊辛玻尔兹曼分布采样；scQuantaVita 实测在 **1025 节点**上相对经典模拟 **≥10⁵ 倍加速**，且真机连续稳定运行 >1 小时。这让我们能训练**真正的全连接互作模型**，而不是经典方法被迫做的稀疏/平均场近似——而"接触 vs 非接触"的判定恰恰依赖于**全连接下的条件独立结构**，近似掉就得不到。

**一句话价值主张**：本路线不是"给传统微生物组分析套个量子壳"，而是把榜题里每一个验收指标（≥10 菌、≥100 代谢物、接触/非接触、广谱/特异、长/短、贡献度 ≥60%）都翻译成一个**可计算、可读出、可湿实验闭环验证**的模型量，并用 CIM 让这套模型在真实群落规模上跑得动。

## 1. 把生物学问题翻译成物理/计算问题

### 1.1 榜题拆解（来自榜单原文）
- 对象：酱香型白酒固态四轮次发酵；乳酸是决定酒体风味与发酵稳定性的关键指标，主要产生菌为酒醅中的**乳酸菌（LAB）**。
- 未知：高温大曲带入酒醅的功能微生物中，**哪些能定植增殖**、对乳酸菌有**抑制作用**、机制是**接触（physical/直接）还是非接触（代谢物介导/间接）**、是**广谱还是特异**、**长效还是短效**，以及各通路对乳酸菌丰度的**相对贡献**。
- 数据：覆盖 ≥4 个工艺节点（丢糟、堆积、入窖、窖内、出窖）× 3 轮次的**宏基因组 + 代谢组**多组学。
- 硬指标：见 §8 逐条映射表。

### 1.2 关键洞察：这是一张"带符号的互作网络"的反演
把每个样本（某节点、某轮次、某平行）看成一个高维系统的一次"快照"：哪些菌高/低、哪些代谢物高/低。**跨越几十上百个样本反复观测，就能反推出系统内部的直接耦合强度。**

- 令系统状态 `s_i ∈ {-1,+1}`（第 i 个菌/代谢物"高/低"），能量
  $$E(s) = -\sum_i h_i s_i - \sum_{i<j} J_{ij}\, s_i s_j,\qquad p(s)\propto e^{-E(s)}.$$
- `J_ij > 0`：倾向同高同低 → **共存/互利**；`J_ij < 0`：倾向此高彼低 → **相互排斥/抑制**。
- `h_i`：第 i 个物种的内在丰度倾向（生态位适配）。

这正是 scQuantaVita 里对单细胞用的同一个能量函数 `E(s)=-Σ J_ij s_i s_j - μ Σ h_i s_i`（见其正文 Fig.1）。**我们要学的 `{J_ij,h_i}` 就是榜题要的"互作网络"本身**，而"关键乳酸菌"节点那一行的负耦合，就是"谁在抑制它、抑制多强"。

### 1.3 为什么"相关"不等于"互作"，以及接触/非接触从何而来
微生物组里最常见的错误是把**相关（correlation）**当**互作（direct interaction）**。若菌 X 产生抑菌代谢物 M、M 压制乳酸菌 LAB，则数据里 `corr(X, LAB) < 0`——但这是**间接**的。玻尔兹曼机/逆 Ising 学的是**直接耦合 J**（等价于偏相关/条件独立，即微生物生态里 SPIEC‑EASI 类图模型的思想），它会把这条 X→M→LAB 的间接相关"扣掉"：

- **接触（直接）效应**：把代谢物也放进模型后，`J[X, LAB]` 仍显著为负 ⇒ X 对 LAB 的抑制**不经由已测代谢物**（细胞接触、竞争、直接分泌未被代谢组捕获的因子）。
- **非接触（代谢物介导）效应**：`J[X, LAB] ≈ 0`，但 `J[X, M] > 0` 且 `J[M, LAB] < 0` ⇒ X 通过 M **间接**压制 LAB。
- **判据是"结构性"的**：只有在**全连接**模型里同时纳入菌与代谢物节点、做直接耦合分离，接触/非接触才可判。平均场/稀疏近似会把弱而关键的中介路径抹平——这就是量子（可训全连接 BM）不可替代的地方。

## 2. 为什么是量子（CIM / 开物）——瓶颈与证据

### 2.1 计算瓶颈：配分函数与阻挫
玻尔兹曼机训练的梯度是经典结果：
$$\frac{\partial \log L}{\partial J_{ij}} = \langle s_i s_j\rangle_{\text{data}} - \langle s_i s_j\rangle_{\text{model}},\quad
\frac{\partial \log L}{\partial h_i} = \langle s_i\rangle_{\text{data}} - \langle s_i\rangle_{\text{model}}.$$
数据相 `⟨·⟩_data` 一次算完；**难的是模型相 `⟨·⟩_model`**——要在 `p(s)∝e^{-E}` 上求期望，涉及 `Z=Σ_s e^{-E(s)}`，`s` 的组合数 `2^N` 随节点数 N 指数增长。真实菌群互作网**稠密且阻挫**（竞争=负环），经典 Gibbs/对比散度在这种能量面上混合极慢、易陷局部。

### 2.2 CIM 的作用：硬件级玻尔兹曼采样
- 相干伊辛机（CIM）以 DOPO 相位作自旋，直接对伊辛模型定义的玻尔兹曼分布采样（`p(s)∝e^{-E_θ(s)/kT}`）。
- **scQuantaVita 的实测证据**（同一支 QBoson×广州国家实验室团队）：
  - 在 38–1025 节点的 BM 网络上验证了 `ln(p_s)` 与 `-E_s/kT` 的强相关（即确为玻尔兹曼采样，其 Fig.2C/2D）。
  - 相对经典 CIM 模拟（laps=1e4~1e5），**最大规模处 ≥10⁵ 倍采样加速**（Fig.2E）。
  - 真机通过光纤环温控与隔振，**连续稳定计算 >1 小时**。
- 结论：CIM 把"负相采样"从指数瓶颈变成硬件常数时间的一步，使**全连接、社区规模**的互作模型第一次可训——这正是榜题需要的粒度。

### 2.3 与经典方法的定位（不是二选一，而是补短板）
| 经典路线 | 局限 | 本路线如何补 |
|---|---|---|
| 相关网络（SparCC/CoNet） | 只给相关、不分直接/间接 | BM 学直接耦合 J → 接触/非接触可判 |
| 图模型（SPIEC‑EASI，稀疏假设） | 稠密竞争网被稀疏化，弱中介路径丢失 | CIM 训全连接 BM，不强加稀疏 |
| gLV 广义 Lotka‑Volterra | 需时序密采、参数病态 | 能量模型用截面数据即可反演，节点时序补充长/短判定 |
| 经典 RBM/对比散度 | 负相采样慢且有偏 | CIM 负相采样 10⁵× 且近热态 |

### 2.4 协作分工与量子的定位
- **经典主体（资深生信合作团队）**：多组学建库、物种/功能注释、常规差异分析与相关网络——这部分经典方法成熟、由生信专家主导完成。
- **量子差异化（玻色量子：CIM + QVAE + BM）**：承担**最难、最有区分度、也最有论文/专利价值**的子问题——从截面多组学**反演直接互作网络**并判定**接触/非接触**。经典相关/平均场在此系统性不足（见 2.3），这正是量子不可替代、能显著提升中标概率与成果新颖性的环节。
- **量子亦作独立验证**：CIM 学到的互作结论可与经典相关网络交叉印证，形成"经典发现 + 量子验证"的双证据链。
- **严谨性（关键）**：我们不采用"看隐变量里哪些特征共现就说它们互作"这类定性直觉（易被质疑不够严格），而是把它升级为**逆 Ising / 直接耦合分析（DCA）**的严格统计物理框架——直接耦合 = 偏相关/条件独立（SPIEC‑EASI 谱系），可量化、可发表、可申专利。这是把"隐变量共现"变成经得起评审推敲的科学方法的关键一步。

## 3. 方法框架（三阶段，逐一对映指标）

> 记号沿用 scQuantaVita：**Boltzmann State Encoding**（把连续组学离散成二值隐层）、**CSP = Cellular System Potential**（细胞系统势 = 该样本隐态的能量）。在菌群语境我们把后者改称 **CSP‑community / 群落系统势**。

### 阶段①　QVAE（玻尔兹曼态编码）：降维 + 抓关键菌群
**输入**：每个样本一条高维向量 = 菌相对丰度（宏基因组，CLR 变换）⊕ 代谢物强度（代谢组，log/z‑score）。
**编码器**：`x → q →` spike‑and‑exponential `→ ζ →` 离散化为二值隐层 `z ∈ {0,1}^D`（scQuantaVita 用 256 维）。straight‑through 估计器保证反向传播可导。
**解码器**：由 `z` 重构丰度，验证隐层保真。
**产出**：
- **关键菌群/代谢物抽取**：找出对"高乳酸菌 vs 低乳酸菌"分离轴载荷最大的特征 → 候选关键微生物（≥10）与关键代谢物（≥100）。→ 对映**指标 1、2 的"物种/代谢物清单"**。
- **群落系统势 CSP‑community**：每个样本隐态的能量 `E(s)`，给出该工艺节点群落的**热力学稳定度**。堆积/入窖过渡态高能（不稳定、正在演替），窖内成熟态低能（稳定）。→ 支撑**"迁移延续与动态演替规律"**（指标 1）。

> 工程上：阶段① 默认用 PyTorch 忠实实现（与 scQuantaVita 一致）；无 GPU/torch 时，`qmicrobiome_tutorial.py` 自动退化为 PCA→二值化，保证 pipeline 能跑通、演示不中断。

### 阶段②　玻尔兹曼机互作层：学耦合网络 {J_ij, h_i}（CIM 采样）
在关键特征（菌 + 代谢物节点，或隐层维度）上训练全连接 BM：
1. 由数据算 `⟨s_i⟩_data, ⟨s_i s_j⟩_data`（一次）。
2. **负相 `⟨·⟩_model` 由 CIM 采样估计**：把当前 `{βh, βJ}` 编码成 QUBO→伊辛矩阵（`kw.conversion.qubo_matrix_to_ising_matrix`），用 `kw.cim.SimulatedCIMOptimizer`（离线模拟）或 `kw.cim.CIMOptimizer`（真机）取一批构型，按能量重要性重加权得期望。
3. 梯度上升更新 `{J,h}` 直到收敛。
**产出**：带符号的互作网络 `J`——第 LAB 行/列即"抑制 LAB 的菌与代谢物及强度"。

### 阶段③　反事实敲除：贡献度量化 + ≥60% 关键菌验证
用训好的能量模型做**in‑silico 敲除**：把某菌节点钳制为"缺失"（`s_X=-1`），用 CIM 重采样，测关键乳酸菌"高丰度"边缘概率 `P(LAB=high)` 的变化 `ΔP_X`。
- 各通路 `ΔP` 归一 → **每条互作通路对乳酸菌丰度的相对贡献度**。
- 按贡献排序，取**累计 ≥60%** 的菌集合 → **待湿实验验证的关键菌**。→ 直接对映**指标 3**。

## 4. 接触/非接触、广谱/特异、长/短——量子可判据（指标 2 的核心）

| 生物学问法 | 模型读出 | 说明 |
|---|---|---|
| **接触（直接）** | 纳入代谢物节点后 `J[X,LAB] < 0` 仍显著 | 抑制不经已测代谢物：接触、竞争、直接分泌 |
| **非接触（介导）** | `J[X,LAB]≈0` 且存在 `J[X,M]>0, J[M,LAB]<0` | X 经代谢物 M 间接压制 LAB |
| **广谱** | `J[X,·]` 对多个物种均显著负 | 抑制谱宽 |
| **特异** | `J[X,·]` 仅对 LAB（或少数）显著负 | 靶向抑制 |
| **长效** | 逐节点/轮次分别拟合 `J`，`J[X,LAB]` 跨节点稳定存在 | 抑制持续 |
| **短效** | `J[X,LAB]` 仅在个别节点出现后消失 | 瞬时抑制 |

> 长/短的时序判定：把"节点×轮次"作为条件，训练**节点条件化的 BM**（或加时序耦合边），比较耦合的跨节点持久性。这也顺带回答"从大曲迁移进酒醅后作用的时空延续性"。

**演示（见 tutorial 输出）**：我们在合成数据里**预埋**一个接触抑制菌 C、一个经代谢物 M 的非接触抑制菌 NC。BM 能正确地：对 C 学出显著负的 `J[C,LAB]`；对 NC 学出 `J[NC,LAB]≈0` 但 `J[NC,M]>0, J[M,LAB]<0`——**尽管原始 `corr(NC,LAB)` 也是负的**。这正是"相关≠互作、量子 BM 能分离接触/非接触"的可复现证据。

## 5. 贡献度量化与湿实验验证闭环（指标 3）

1. **模型内贡献度**：敲除重采样得 `ΔP_X`，归一化为贡献占比；累计 ≥60% 得关键菌集。小规模用精确枚举校验，大规模用 CIM 采样——tutorial 中展示两者在小问题上一致，从而信任大规模只能靠 CIM 的那部分。
2. **建立验证方法（榜题明确要求"建立科学验证方法"）**：
   - **接触 vs 非接触的湿实验**：Transwell/膜隔共培养（隔绝接触、保留代谢物）对比直接共培养 → 验证模型给的接触/非接触标签。
   - **非接触通路验证**：关键代谢物 M 的**外源添加（spike‑in）/敲低**，看 LAB 抑制是否复现。
   - **关键菌验证**：候选菌**回接酒醅微宇宙/回添大曲**，测 LAB 丰度变化，比对模型预测的 `ΔP`。
   - **后验贡献率 ≥60%**：以湿实验实测的 LAB 丰度变化拟合"实测贡献"，要求筛出的关键菌集合后验贡献率 ≥60%（榜题指标 3 原文）。

## 6. 数据与工程方案

- **样本设计**：丢糟/堆积/入窖/窖内(多时点)/出窖 × 3 轮次 × 生物学平行，配套大曲样本；宏基因组（物种+功能基因）+ 代谢组（≥100 代谢物）；扩增子（OTU/ASV）亦可纳入。注意样本量：单分组乐观数十~数百、悲观仅数条——**样本偏少时优先用宏基因组/代谢组，并加强生物学平行**，否则 QVAE 相对 PCA/PCoA 的增益不足。
- **预处理（对齐现有实践）**：扩增子/宏基因组相对丰度高维稀疏——先**筛选高可变特征（high-variance OTU/物种）**再 CLR/log/scale（这与内部 QBM‑VAE 实战经验一致，效果显著优于 raw count）；代谢组 log/Pareto scaling；批次（轮次/窖池）用隐层对齐（scQuantaVita 已证明能量约束隐层在 batch integration 上 SOTA，正是应对扩增子批次的优势）。
- **节点构建**：菌节点 + 代谢物节点 + 功能基因通路节点（可选），构成异质图；BM 在其上全连接。
- **规模与硬件**：关键子网数十~数百节点，正是 CIM 的甜区（scQuantaVita 已验证到 1025）。开物 SDK：`kw.cim.SimulatedCIMOptimizer` 离线 CIM 模拟即可跑通全流程；`kw.cim.CIMOptimizer(task_name=..., project_no=...)` 提交**真·相干伊辛机**做规模化关键步。许可：`kw.license.init(user_id, sdk_code)`（platform.qboson.com 免费注册）。
- **实现基座**：QVAE/离散隐层用内部 `DVAE_RBM`（Discrete VAE + RBM，可处理连续组学）为底座，优于 kaiwu‑pytorch‑plugin 里较原始、只吃二值输入的 QVAE；互作层与采样按本报告接开物 CIM。
- **可复现**：全流程离线可跑（reference 后端），CIM 真机为一行切换（采用与 `QDock/kw_backend.py` 同构的 kaiwu/reference 双后端设计）。

## 8. 里程碑、风险与新颖性（对映指标 4）

**三年里程碑（总投入 ≤300 万、周期 ≤3 年）**
- **Y1**：多组学数据库与预处理管线；QVAE 关键菌/代谢物清单（指标 1）；离线 BM 原型（本 tutorial 的规模化版）。
- **Y2**：CIM 真机训练全连接互作网；接触/非接触、广谱/特异、长/短判定（指标 2）；首篇方法学 SCI 投稿。
- **Y3**：反事实贡献度 + 湿实验闭环，锁定后验 ≥60% 关键菌（指标 3）；第二篇应用 SCI；发明专利（量子‑经典混合的菌群互作贡献度量化方法）。

**风险与缓解**
- *数据稀疏/批次强*：能量约束隐层对 batch 更稳（scQuantaVita 证据）；增加平行与节点。
- *代谢组未覆盖真实中介物* → 接触/非接触会误判：以 transwell 湿实验交叉校验，必要时补靶向代谢组。
- *CIM 采样非严格热态* → 负相有偏：用 `e^{-βE}` 重要性重加权校正（tutorial 已实现），并在小问题上与精确枚举对表。
- *真机排队/额度*：离线 CIM 模拟器 `SimulatedCIMOptimizer` 先行，真机 `CIMOptimizer` 只用于规模化关键步。

**新颖性（支撑 SCI×2 + 专利×1）**
1. **首个把量子‑经典混合玻尔兹曼机用于发酵微生物组互作反演**的框架（scQuantaVita 是单细胞，本工作是群落 + 代谢物异质图，且给出接触/非接触判据）。
2. **CSP‑community（群落系统势）**：可量化群落热力学稳定度/演替方向的新指标（scQuantaVita 的 CSP 在群落语境的原创扩展）。
3. **反事实敲除 + CIM 重采样的贡献度量化法**：可申请发明专利。

## 9. 代码 tutorial 说明

配套 `qmicrobiome_tutorial.py`（`# %%` 分块，可直接转 Jupyter/Colab）演示端到端：
1. 生成**预埋真值**的合成多组学数据（含接触抑制菌 C、非接触抑制菌 NC、抑菌代谢物 M、关键乳酸菌 LAB）。
2. **阶段① QVAE**（torch 忠实版 / numpy‑PCA 退化版）→ 关键菌/代谢物 + CSP‑community。
3. **阶段②** 用 **Kaiwu** 负相采样训练玻尔兹曼机，学出 `J,h`；在小问题上与**精确枚举**对表，证明 Kaiwu 估计可信。
4. **接触/非接触、广谱/特异**从 `J` 读出，复现"corr(NC,LAB)<0 但 J[NC,LAB]≈0（介导）"。
5. **阶段③** 反事实敲除量化各通路贡献度，输出累计 **≥60%** 关键菌集。

运行：`python qmicrobiome_tutorial.py`（默认 reference 后端，无需许可）；装好开物后设 `BACKEND="kaiwu"` 并 `kw.license.init(...)` 即切真·CIM。
